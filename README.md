# Gyana Ranjan Behera - Cybersecurity Portfolio

A professional, high-performance cybersecurity portfolio website. Designed with a hacker-aesthetic, featuring glassmorphism, terminal-style interfaces, and smooth interactive elements.

![Portfolio Preview](assets/images/profile.png)

## 🚀 Live Demo
View the live site here: [Gyanaranjan2234.github.io/Portfolio](https://Gyanaranjan2234.github.io/Portfolio)

## ✨ Features
- **Hacker Aesthetic**: Custom dark theme with neon accents and terminal window visuals.
- **VAPT Focus**: Highlighted skills in Vulnerability Assessment and Penetration Testing.
- **Interactive Terminal**: An animated terminal window that introduces the developer.
- **Modern UI/UX**:
  - Glassmorphism effects.
  - Smooth scroll animations (AOS).
  - Dynamic typing effects (Typed.js).
- **Secure Contact Form**: Integrated with EmailJS for direct communication without a backend.
- **Performance Optimized**: Built for speed and responsiveness across devices.
- **SEO Ready**: Semantic HTML5 with meta/Open Graph optimization.

## 🛠️ Tech Stack
- **Languages**: HTML5, CSS3, Vanilla JavaScript
- **Libraries**: AOS.js (Animations), Typed.js (Typing Effects), Font Awesome 6, EmailJS SDK
- **Fonts**: Space Grotesk, JetBrains Mono
- **Deployment**: GitHub Pages / Vercel / Netlify

## ⚙️ Local Setup
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Gyanaranjan2234/Portfolio.git
   cd Portfolio
   ```
2. **Run a local server**:
   ```bash
   # Using Python
   python -m http.server 8000
   ```
3. Open your browser at `http://localhost:8000`.
