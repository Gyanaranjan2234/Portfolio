- **Hacker Aesthetic**: Custom dark theme with neon accents and terminal window visuals.

## ⚙️ Local Setup
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Gyanaranjan2234/Portfolio.git
   cd Portfolio
   ```
2. **Configure EmailJS**:
   - Rename `js/config.example.js` to `js/config.js`.
   - Update with your `serviceId`, `templateId`, and `publicKey`.
3. **Run a local server**:
   ```bash
   # Using Python
   python -m http.server 8000
   ```
4. Open your browser at `http://localhost:8000`.

## 🛡️ Security Best Practices
This project follows professional standards by keeping sensitive credentials (EmailJS keys) in a separate `config.js` file that is **excluded from version control**. This prevents accidental exposure of API tokens on public repositories.
