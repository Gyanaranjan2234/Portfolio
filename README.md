Personal cybersecurity portfolio website of Gyana Ranjan Behera.

## Tech Stack
- HTML, CSS, JavaScript (Vanilla)

## File Structure
```text
Portfolio/

├── script.js           # Core JavaScript logic
├── config.js           # Environment configurations
├── config.example.js   # Example configurations
├── .gitignore          # Git ignore rules

├── js/                 # Additional JavaScript modules
└── scripts/            # Python utility and maintenance scripts
```
