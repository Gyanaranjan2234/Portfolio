### ✅ STEP 4: Deploy Online (Choose One)

#### **EASIEST: Netlify (Recommended for Beginners)**

1. Go to https://netlify.com
2. Sign up with GitHub
3. Click "New site from Git" OR drag & drop your Portfolio folder
4. Wait 1-2 minutes
5. You get a live URL! 🎉

Time: **2-5 minutes**

#### **BEST: GitHub Pages (For Git Users)**

1. Create GitHub account: https://github.com
2. Create new repository named `portfolio`
3. Upload your files (using Git or web interface)
4. Go to Settings → Pages
5. Enable GitHub Pages
6. Site available at: `yourusername.github.io/portfolio` 🎉

Time: **10 minutes**

#### **FAST: Vercel**

1. Go to https://vercel.com
2. Sign up with GitHub
3. Import your repository
4. Click Deploy
5. You get a live URL! 🎉

Time: **3 minutes**

✅ **Done:** Your portfolio is now online!
